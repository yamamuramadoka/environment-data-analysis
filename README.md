# Environment Data Analysis

## Overview

This project analyzes environmental data collected in a classroom.

## Objective

To identify factors affecting indoor environmental quality.

## Tools

- R
- ggplot2
- dplyr

## Analysis

- Data Cleaning
- Data Visualization
- Correlation Analysis
- Generalized Additive Model (GAM)

## Result

Noise level was used as a proxy for human activity.

## Skills

- Data Analysis
- Data Visualization
- Hypothesis Testing
